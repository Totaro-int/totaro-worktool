## 문서 작성 가이드

새로운 문서를 추가할 때는 다음 형식을 따르십시오:

```markdown

> **작성일:** YYYY-MM-DD
> **버전:** X.Y.Z

## 개요

간단한 소개

## 상세 내용

...

## 변경 이력

| 날짜       | 버전  | 변경 사항 |
| ---------- | ----- | --------- |
| YYYY-MM-DD | X.Y.Z | 초기 작성 |
```
