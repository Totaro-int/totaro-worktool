# 팀원 설치 가이드 — Claude 작업 자동 기록

본인이 Claude로 한 작업이 **업무 툴(claude_logs)에 자동으로 쌓이게** 하는 1회 설치.
요약은 태준 Mac이 중앙에서 처리하니, 각자는 "설치"만 하면 됩니다.

## 어느 방법으로?

| 이렇게 쓰면                      | 방법                    |
| -------------------------------- | ----------------------- |
| **Claude Code(터미널)** 를 쓴다  | **A. 개발자 설치**      |
| **claude.ai 데스크탑 앱**만 쓴다 | **B. 무설치(더블클릭)** |

둘 다 쓰면 A 로 하면 챗까지 같이 잡힙니다.

---

## A. 개발자 설치 (Claude Code 사용자)

**사전 준비 (1회):**

- Node.js — https://nodejs.org 에서 LTS 설치
- Claude Code CLI — 터미널에 `npm i -g @anthropic-ai/claude-code` 후 `claude` 로 로그인

**설치:**

```bash
# 1) 레포 받기 (고정된 위치에 두세요 — 나중에 옮기면 경로가 깨집니다)
git clone https://github.com/Totaro-int/totaro-worktool.git
cd totaro-worktool
#   (이미 받아놨으면) git pull

# 2) 본인 이름으로 설치 한 줄
bash scripts/setup-logging.sh "이름"     # 예: "승주"
```

설치되는 것:

- **SessionEnd + Stop 훅** — Claude Code/코워크 작업을 세션 끝날 때 + 긴 세션은 3시간마다 기록
- **launchd 에이전트** — claude.ai 챗을 로그인할 때마다 + 매일 새벽 수집

이후 자동입니다. 컴퓨터를 껐다 켜도 따라잡습니다.

---

## B. 무설치 (claude.ai 앱만 쓰는 분 / 비개발자)

1. 태준에게 **`totaro-chat-setup.zip`** 을 받습니다.
2. 압축을 풀고 **`설치.command` 를 더블클릭**합니다.
3. "확인되지 않은 개발자" 경고가 뜨면 → 파일을 **우클릭(또는 Control+클릭) → 열기 → 열기**.
4. 이름을 입력하고 **설치**를 누르면 끝.

터미널·Node·git 전부 필요 없습니다. 이후 컴퓨터를 켤 때마다 자동 수집됩니다.

---

## 잘 됐는지 확인

- 작업을 한 뒤(또는 세션을 한 번 끝낸 뒤), 업무 툴의 **Claude 작업 기록** 페이지(`/claude-log`)에
  본인 이름의 기록이 뜨면 성공입니다.
- A 방법은 **세션을 끝내거나 3시간이 지나면** 기록이 올라옵니다 (실시간 아님).

문제 있으면 태준에게 문의. (`~/.totaro/` 폴더의 로그 파일로 진단 가능)
